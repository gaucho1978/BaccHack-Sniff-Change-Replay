**BaccHack - Ethernet Packet Player and Changer**
    
This application uses Pcap (NDIS driver) to replay a previously recorded pcap wireshark file. The packets can be replayed with their original timing or different timing. The packets can be replayed as they are, or some field can be changed (like, ip, mac, port). the binary is included in the repository in order for you to be ready to be used. if you don't have wireshark on your computer, first thing to do is to install pcap install program, included in the bin/debug/leggimi folder. Enjoy.
